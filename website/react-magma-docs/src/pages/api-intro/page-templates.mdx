# Page Templates (work in progress)

Get started on your project more quickly with these ready-made page templates.

## Full-width Template

This template is ideal for applications that need as much of the screen as possible.

<figure>
  <img src="../../images/layout/template-full-width.png" alt=" " />
</figure>

```noRender
import React from 'react';
import styled from '@emotion/styled';
import { Header } from '@cengage-patterns/header';
import { magma, useMediaQuery } from 'react-magma-dom';

const Container = styled.div`
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: auto 1fr;
  grid-template-areas:
    'header header'
    'content content';
`;

const StyledHeader = styled(Header)`
  grid-area: header;
`;

const Content = styled.div`
  grid-area: content;
  max-width: 1600px;
  justify-self: center;
  width: calc(100% - 48px);
`;

export function Example() {

  const isSmallerScreen = useMediaQuery(
    `(max-width:${magma.breakpoints.small}px)`
  );

  return (
    <Container>
      <StyledHeader isCompact={isSmallerScreen}></StyledHeader>
      <Content></Content>
    </Container>
  );
}
```

## Global Header with Sidebar

This template includes a screen region for a left sidebar and includes all of the necessary responsive behavior.

<figure>
  <img src="../../images/layout/template-sidebar-app-header.png" alt=" " />
</figure>

```noRender
import React from 'react';
import styled from '@emotion/styled';
import { Header } from '@cengage-patterns/header';
import { magma, useMediaQuery } from 'react-magma-dom';

const Container = styled.div`
  display: grid;
  grid-template-columns: 240px 1fr;
  grid-template-rows: auto 1fr;
  grid-template-areas:
    'sidebar header'
    'sidebar content';
`;

const StyledHeader = styled(Header)`
  grid-area: header;
`;

const Sidebar = styled.div`
  grid-area: sidebar;
`;

const Content = styled.div`
  grid-area: content;
  max-width: 1600px;
  width:100%;
  justify-self:center;
`;

export function Example() {

  const isSmallerScreen = useMediaQuery(
    `(max-width:${magma.breakpoints.small}px)`
  );

  return (
    <Container>
      <StyledHeader isCompact={isSmallerScreen}></StyledHeader>
      <Sidebar></Sidebar>
      <Content></Content>
    </Container>
  );
}
```
