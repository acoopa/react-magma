<DocsHeading type="api">ButtonGroup</DocsHeading>

Layout component for Buttons

## Default

```typescript
import React from 'react';
import { ButtonGroup, Button } from 'react-magma-dom';

export function Example() {
  return (
    <ButtonGroup>
      <Button>1</Button>
      <Button>2</Button>
      <Button>3</Button>
    </ButtonGroup>
  );
}
```

## Horizontal - Apart

The `apart` prop should only be used with the `horizontal` orientation.

```typescript
import React from 'react';
import { ButtonGroup, Button, ButtonGroupOrientation } from 'react-magma-dom';

export function Example() {
  return (
    <ButtonGroup
      orientation={ButtonGroupOrientation.horizontal}
      alignment="apart"
    >
      <Button>1</Button>
      <Button>2</Button>
      <Button>3</Button>
    </ButtonGroup>
  );
}
```

The `noSpace` prop should only be used with the `horizontal` orientation.

## Small Screen Sizes

```
import React from 'react';
import { ButtonGroup, Button, ButtonGroupOrientation, BreakpointsContainer, Breakpoint, BreakpointScreenSize } from 'react-magma-dom';

export function Example() {
  return (
    <BreakpointsContainer>
      <Breakpoint screenSize={BreakpointScreenSize.xs}>
        <ButtonGroup orientation={ButtonGroupOrientation.vertical}>
            <Button>1</Button>
            <Button>2</Button>
            <Button>3</Button>
            <Button>4</Button>
            <Button>5</Button>
          </ButtonGroup>
      </Breakpoint>

      <Breakpoint screenSize={BreakpointScreenSize.small}>
        <ButtonGroup>
          <Button>1</Button>
          <Button>2</Button>
          <Button>3</Button>
          <Button>4</Button>
          <Button>5</Button>
        </ButtonGroup>
      </Breakpoint>
    </BreakpointsContainer>
  );
}
```
